# 🎉 Shree Shyam Decoration & Events

A modern **MERN Stack** platform built for **Shree Shyam Decoration & Events** to manage and showcase decoration services, event packages, gallery media, products, and customer inquiries.

The project is designed as a **real-world production application** with a dedicated customer-facing website and a separate admin portal.

---

## 📌 Project Structure

```text
shree-shyam-decoration-events/

├── client-public/      # Customer Website
├── client-admin/       # Admin Dashboard
├── server/             # REST API & Backend
└── docs/               # Project Documentation
```
